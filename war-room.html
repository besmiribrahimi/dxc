About ENTRENCHED WW1

Core Gameplay

Loosely based on World War 1, with Entente and Central Powers trench warfare.

Features 9 classes and modes like frontline attack/defense, conquest, and domination.

Latest Public Update

Version v1.12.1 Class Mastery update includes Neuve Chapelle 1915 and mastery charms.

Also lists map vote boost and gamepass gifting features.

Technical Snapshot

Genre: Shooter (Deathmatch Shooter). Server size: 40 players.

Playable on PC, Mobile, and Console. Created 8/18/2019, updated 3/28/2026.

Momentum Points Model

Faction score is calculated per player as (Level * 12) + (K/D * 18) + Rank Bonus, then totaled by faction to show current frontline pressure.
